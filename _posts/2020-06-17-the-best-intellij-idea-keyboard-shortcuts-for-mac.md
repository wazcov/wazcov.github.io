IntelliJ IDEA is one of my favourite IDEs, and a key reason for that is the great shortcuts that makes my life easier when developing software. 

Here are a few of the most useful ones:

**Control + Option + O** &#8211; Optimises Imports (removes imports you don&#8217;t need)

**Command + Option + L** &#8211; Formats your code

**Option + Enter** &#8211; Import (if on a red highlighted piece of code)

**Command + /** &#8211; Line comments all selected code

**Command + Shift + /** &#8211; Block comment for all selected code

**Command + J** &#8211; Brings up live templates  
(then you can type for example m &#8211; and in Python it will auto generate the main method)
